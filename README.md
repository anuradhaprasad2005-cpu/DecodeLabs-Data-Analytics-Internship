### 📌 Project 2 – Exploratory Data Analysis (EDA)

**Objective:**  
Perform Exploratory Data Analysis on an e-commerce dataset to identify business trends, revenue drivers, and operational challenges.

**Analysis Performed:**
- Dataset Profiling
- Descriptive Statistics
- Outlier Detection (IQR Method)
- Revenue Analysis
- Product Category Analysis
- Referral Channel Analysis
- Payment Method Analysis
- Order Status Analysis
- Correlation Analysis
- Business Insights & Recommendations

**Key Findings:**
- Total Revenue: ₹12,64,762
- Average Order Value: ₹1,053.97
- Chair and Printer generated the highest revenue
- Instagram was the highest-performing referral channel
- Credit Card users had the highest average order value
- 41.4% of orders were Cancelled or Returned

**Outcome:**  
Generated actionable insights and recommendations to support data-driven decision-making.
